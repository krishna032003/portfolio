# Phase 3 Research: Premium Portfolio Design for AI Engineers

## Modern Trends (2026)
- **Exaggerated Typography**: Using `text-5xl` to `text-7xl` for hero headlines to create instant impact.
- **Agentic Previews**: Incorporating simulated terminal or code-typing effects to signal AI expertise.
- **Bento Grids**: Organizing skills and achievements into modular, glassmorphic blocks.
- **Staggered Motion**: Using Framer Motion to reveal elements sequentially (Title -> Tagline -> CTA).

## Layout Decisions
- **Hero**: Centered or split layout with a prominent "typing" effect for the tagline or a code block.
- **About**: Split layout: Left (Bio text), Right (Stylized Education/Quick Facts in a Glass Card).
- **Skills**: Bento Grid layout categorizing skills into Frontend, Backend, AI/ML, and Tools.
- **Achievements**: Card-based grid highlighting leadership roles and hackathon wins.

## Reference Styles
- **Colors**: Primary Background (#0f1115), Accent (#00f3ff - Neon Electric Blue).
- **Components**: `GlassCard`, `SectionWrapper`, `SectionHeading`, `Button`.
