# Phase 4 Research: Interactive Project Showcases & Contact

## Interactive Project Cards
- **3D Tilt Effect**: Using `framer-motion`'s `useMotionValue` and `useTransform` to create a perspective tilt that follows the mouse.
- **Dynamic Borders**: Implementing "Glow Borders" that light up when hovered, matching the neon electric blue theme.
- **Content Reveal**: Using staggered entrance for tech stack badges and "View Code/Demo" buttons on hover.

## Contact Section Trends
- **Minimalist Directness**: Moving away from complex forms (unless specifically requested) toward high-visibility email and social links with premium glassmorphism.
- **Social Proof**: Integrating GitHub/LeetCode profile links with stats-aware tooltips.

## Layout Decisions
- **Grid**: A responsive grid that transitions from 1 column (mobile) to 2-3 columns (desktop).
- **Cards**: Fixed aspect ratio with a "featured image" placeholder or abstract AI-generated pattern (since real screenshots aren't provided).
